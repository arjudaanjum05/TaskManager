# Task manager

## Features:

USER and ADMIN are able to:

- Create project
- Get all projects
- Delete project
- Create a task
- Edit task (change description, status, due date). Closed tasks cannot be edited.
- Get task
- Search tasks
  ◦ Get all by project
  ◦ Get expired tasks
  ◦ By status
  ADMIN additionally should be able to:
- Get all tasks by user
- Get all projects by user

## Built With

- Spring Boot
- Spring Security
- MySQL database
- Maven
- Thymeleaf
- Bootstrap
- jQuery
